1) Открыть папку в проводнике 

2) Закрыть решение

3) Будет папка git, туда кидаем проект

4) Кликаем правой кнопкой мыши в пустое пространство в папке git и в меню выбираем 

Git bash here

5) вводим команду для добавления в 1 раз,

git add .

6) git commit -m "Add_project"

7) git push -u origin 

(если ошибка то: git push --force origin)
